# راهنمای سریع پنل مدیریت وب

## نصب سریع

```bash
# نصب وابستگی‌ها
pip install Flask Flask-Login Werkzeug

# یا نصب همه وابستگی‌ها
pip install -r requirements.txt
```

## اجرا

```bash
# Windows
run_web.bat

# Linux/Mac
./run_web.sh

# یا مستقیماً
python -m web.app
```

## دسترسی

1. مرورگر را باز کنید و به آدرس زیر بروید:
   ```
   http://localhost:5000
   ```

2. با اطلاعات زیر وارد شوید:
   - **نام کاربری:** `admin`
   - **رمز عبور:** مقدار `ADMIN_PASSWORD` در فایل `.env` (اگر تنظیم نکرده‌اید، پیش‌فرض: `admin123`)

## تنظیم رمز عبور ادمین

در فایل `.env` اضافه کنید:
```env
ADMIN_PASSWORD=رمز-عبور-قوی-شما
WEB_SECRET_KEY=یک-کلید-تصادفی-قوی-برای-امنیت
```

## ویژگی‌های پنل

✅ **داشبورد:** آمار کامل سیستم با نمودارها  
✅ **مدیریت کاربران:** مشاهده، جستجو، قفل/باز کردن  
✅ **مدیریت حساب‌ها:** مشاهده، فعال/غیرفعال کردن  
✅ **مدیریت تراکنش‌ها:** مشاهده، فیلتر و جستجو  
✅ **رابط کاربری فارسی:** طراحی مدرن و واکنش‌گرا  
✅ **بروزرسانی خودکار:** هر 30 ثانیه

## نکات مهم

- پنل وب می‌تواند همزمان با بات تلگرام اجرا شود
- هر دو از همان دیتابیس استفاده می‌کنند
- برای production، حتماً HTTPS و رمزهای قوی استفاده کنید
